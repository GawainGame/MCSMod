# 暖雪 内置修改器 

开源地址：https://git.aoe.top/Terry13143/SW_ScriptTrainer

下载地址：https://mod.3dmgame.com/mod/181716

### 安装方法
- 先安装 [BepInEx](https://github.com/BepInEx/BepInEx/releases/tag/v5.4.5 "BepInEx")
- 然后将ScriptTrainer.dll放到WarmSnow\BepInEx\plugins中；
- 如果没有“plugins”文件夹自己创建一个
- 进游戏按F9呼出修改器菜单

### 功能
- 开启官方作弊
- 回满血量
- 修改蓝魂
- 修改红魂
- 获取道具（最高品质
- 获取武器（最高品质



### 作弊功能

- TAB 暂停游戏
- F1 起死回生 （按一下会给红色的那玩意
- F2 解锁全部成就  
- F3 完成支线
- F4 添加移速和攻速
- F6 获取所有道具
- F7 获取所有武器
- F8 清空创建的东西
- H 获取蓝魂
- J 获取红混
- O 获取两个宗派技能
- U 释放冰柱 冻住敌人的行动
- I 释放个不知道是啥的东西 秒杀全屏
- 还有一大堆没看懂的 在 PlayerAnimControl.Update() 函数里面 有兴趣的可以自己去看
